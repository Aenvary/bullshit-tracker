# Bullshit Tracker

An expense tracker that judges your spending. Categorize each purchase as **needed**, **worth it**, or **bullshit** — then review later and watch the regret tag light up when you change your mind.

## Features

- Add expenses with description, type, amount, date, and category
- Tap to cycle category (the "regret button")
- Asymmetric regret tag — only fires when you reclassify something *to* bullshit
- Two-tap delete (no accidents)
- Per-category and grand totals
- "Days since last bullshit" streak counter
- Persists locally via localStorage
- Export to JSON
- Mobile-first dark mode

## Stack

Vanilla HTML, CSS, JavaScript. No build step, no framework, no dependencies.

## Run

Open `index.html` in a browser. That's it.
